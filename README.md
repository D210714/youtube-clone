# 🎥 YouTube Clone

A responsive YouTube homepage clone built using **HTML5** and **CSS3**. This project recreates the basic YouTube user interface, including the navigation bar, sidebar, video grid layout, search section, notification icons, and responsive design.

## 🚀 Features

* Responsive YouTube-style layout
* Fixed header navigation
* Sidebar menu
* Search bar with voice search button
* Video thumbnail cards
* Channel profile images
* Notification badge
* Hover tooltips
* Mobile, tablet, and desktop responsiveness

## 🛠️ Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries
